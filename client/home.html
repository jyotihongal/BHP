House Price Prediction

House prices are an important reflection of the economy, and housing price ranges are of great interest for both the buyers and sellers.In this project, house prices will be predicted using the given explanatory variables that cover many aspects of residential houses.The dataset will be put on a step by step process before being used to build the model and the system.The goal of this project is to create a model using regression algorithm that is able to accurately estimate the price of the house with the given features.
